# 📊 OBE Management System

An **Outcome-Based Education (OBE) Management System** built using Flask to manage, track, and analyze Course Outcomes (CO), Program Outcomes (PO), and student performance.

---

## 🚀 Features

### 👨‍🏫 Faculty Module
- Add / manage Course Outcomes (CO)
- Map CO with assessments
- Enter marks for direct evaluation
- View CO attainment reports


### 📊 Analytics Dashboard
- CO Attainment visualization
- Bar chart analysis
- Radar chart analysis
